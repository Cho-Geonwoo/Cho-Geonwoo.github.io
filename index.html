Geonwoo Cho

Hi! I am an undergraduate student at GIST, where I work with Sundong Kim.

I’m interested in developing open-ended curricula, scaling reinforcement learning algorithms, and integrating causal inference into reinforcement learning. To this end, my research focuses on unsupervised reinforcement learning and unsupervised environment design.

Previously, I worked as a machine learning software engineer at Match Group/Tinder, Hyperconnect LLC, and several other companies, gaining hands-on experience with deploying machine learning in production.

Causal-paced Deep Reinforcement Learning
Geonwoo Cho, Jaegyun Im, Doyoon Kim, Sundong Kim
Preprint

Preprint
Website

AMPED: Adaptive Multi-objective Projection for balancing Exploration and skill Diversification
Geonwoo Cho*, Jaemoon Lee*, Jaegyun Im, Subi Lee, Jihwan Lee, Sundong Kim
Preprint
Paper / Website / Code

We introduce AMPED, a framework for skill‐based reinforcement learning that simultaneously maximizes state coverage and skill diversity through several carefully designed components.
